### Summary: 
The purpose of the Music Search is to seamlessly retrieve the lyrics of a song without any ads.

### Technology Used: 
* JavaScript
* HTML
* CSS
* lyrics.ovh API.

### User Guide: 

On the landing page, you can search for a song or a singer to if you would like to know their songs' lyrics.
